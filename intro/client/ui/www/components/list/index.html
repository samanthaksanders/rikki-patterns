Base production

Styled

Language Sample
Jinja2 & Nunjucks 
{% import "rikki.html" as rikki %}
rikki.list(
    list_type="styled", 
    elements=["The list component is really simple.","Yet so important.","Lists have content \"elements\", and a type."]
)
HTML 
<ul class="list-styled">
    <li>The list component is really simple.</li>
    <li>Yet so important.</li>
    <li>Lists have content "elements", and a type.</li>
</ul>
